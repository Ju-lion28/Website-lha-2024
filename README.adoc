= Website LHA 2024

KRYPTIK competition

== Build process

NOTE: Make sure to be in the website's [.underline]#root directory.#

[arabic]
. Install https://nodejs.org/en[NodeJS]
. Install the local packages:
+
[source,sh]
----
npm install .
----
. For *Windows* run:
+
[source,sh]
----
npm run build-win
----
+
For *MacOS* or *Linux* run:
+
[source,sh]
----
npm run build
----
