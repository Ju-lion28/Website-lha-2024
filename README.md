# Website LHA 2024

[Website](https://laurenhill2024.crcrobotics.com/) submission for CRC KRYPTIK competition

## How to build

1. Install [NodeJS](https://nodejs.org/en)
2. Install the local packages:

    ```sh
    npm install .
    ```

3. For Windows run:

    ```sh
    npm run build-win
    ```

    For MacOS or Linux run:

    ```sh
    npm run build
    ```
